# 📝 Descrição do projeto

Site front end V2 da RESTful API de estudos.
Permite um CRUD de categorias associadas a tarefas, com autenticação de usuários.

# 🚀 Funcionalidades

## Categorias

**Adicionar** <br/>
**Editar** <br/>
**Remover** 

## Tarefas

**Adicionar** <br/>
**Editar** <br/>
**Remover** <br/>

## Usuarios

**Registrar** <br/>
**Logar** <br/>
**Mudar senha**
